# Library Management System

## 📌 Project Description
Library Management System is a Java-based console application that allows users to manage books in a simple library system. The system supports adding, viewing, borrowing, returning, and searching books efficiently using Object-Oriented Programming (OOP) concepts.

---

## 🎯 Features
- Add new books to the system
- View all available books
- Borrow books (update status to "Borrowed")
- Return books (update status to "Available")
- Search books by title
- Track book availability status

---

## 🛠️ Technologies Used
- Java
- Object-Oriented Programming (OOP)
- ArrayList
- Scanner (User Input)

---

## 💡 How It Works
1. User runs the program.
2. Main menu is displayed.
3. User selects an option:
   - View Books
   - Add Book
   - Borrow Book
   - Return Book
   - Search Book
4. System processes user input.
5. Book data is stored and managed using ArrayList.
6. Status of books updates dynamically.
